# 🐦 Floppy William 🎂

A silly birthday-themed Flappy Bird clone. Flap through the cakes, grab the presents, don't crash!

## 🎮 Play it now

### 👉 **[Play Floppy William](https://s1msen.github.io/floppy-william/)** 👈

Works on desktop and mobile. No install needed.

## How to play

- **SPACE / Click / Tap** to flap
- Dodge the cakes 🎂
- Collect presents 🎁 for bonus points
- Your best score is saved locally

## Tech

Pure HTML / CSS / JavaScript — no build step, no dependencies. Hosted on GitHub Pages.

## Run locally

```bash
git clone https://github.com/S1msen/floppy-william.git
cd floppy-william
# Just open index.html in a browser
xdg-open index.html   # Linux
open index.html       # macOS
start index.html      # Windows
```
